p5-server, and the P5 Server Visual Studio Code extension, automatically include these libraries if a sketch refers to functions or classes that they define, and the sketch does not include its own definitions of these variables.

Core Libraries

The “Core” libraries from the official p5.js Libraries page.

This does not include the p5.accessibility library because, as noted on its home page, it is obsolete.
